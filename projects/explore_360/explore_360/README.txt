READ ME FILE FOR EXPLORE_360

- establishments.sql:
Contains the database construction of all establishments available on our website.

- reviews.sql:
Contains the database construction of the reviews table. This database table is defaulted to have no reviews until populated by users.

- showEstablishments.php:
A file used for testing which displays all establishments found on the website (in table form), along with their ratings, addresses, names, and categories.

- showReviews.php:
A file used for testing which displays all reviews left by users for an establishment, along with their star ratings.



FRONT-END FOLDER: Contains the functional UI components and each of the 10 pages of the site, along with CSS styling.


- home.php:
Contains HTML web page elements for the HOME page; the main page of our website. Also holds php interaction with database for search functionality.

- hotels.html:
Contains HTML web page elements for the HOTELS page.

- restaurants.html:
Contains HTML web page elements for the RESTAURANTS page.

- shopping.html:
Contains HTML web page elements for the SHOPPING page.

- things to do.html:
Contains HTML web page elements for the THINGS TO DO page.

- events.html:
Contains HTML web page elements for the EVENTS page.

- guidelines.html:
Contains HTML web page elements for the GUIDLINES page.

- history.html:
Contains HTML web page elements for the HISTORY page.

- reviews.php:
Contains HTML web page elements for the REVIEWS page. Contains various jQuery AJAX components and allows users to read and create reviews for establishments.

- displayReviews.php:
Holds relevant source code for the display of reviews as they are updated by the user - works alongside reviews.php for the dynamic display of the reviews.sql database table.

- contacts.html:
Contains HTML web page elements for the CONTACTS page.

Styles.css, styles2.css, styles3.css:
Various stylesheets that are referenced throughout the php and html documents listed above. These 3 files rule the overall design and aesthetic of EXPLORE 360.
